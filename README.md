# MegaLotto

A Ruby gem to generate lottery drawings.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'mega_lotto'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mega_lotto

## Usage

```ruby
MegaLotto::Drawing.new.draw # => [23, 2, 12, 4, 3, 26]
```
